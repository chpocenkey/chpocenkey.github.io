思路

先使用 checksec 检查

发现是 32 位小端序，开启了 NX 保护

使用 IDA Pro 查看，发现有非常多的函数，应该是静态链接，没有用到动态链接，此时可以通过拼接整个程序中的 gadget 实现 getshell 了

可以看到 main 函数中有一堆被 nop 掉的内容，但是实际上不影响做题，此外还有一个函数 overflow ，非常明显的提示

overflow 函数非常简单且明显的栈溢出，一个 gets 函数非常敷衍

查找一下能利用的函数，发现没有 system 等可以直接利用的函数

此时可以使用 ROPgadget 的一个功能生成 ropchain 实现破解

得到结果

将生成的内容通过 pwntools 发送出去就能实现 getshell 了（记得溢出的数量需要自己填充）

exp

总结

  • 其实就是直接全部使用 gadget 的 ROP，但是不得不说，确实是又学了一招
